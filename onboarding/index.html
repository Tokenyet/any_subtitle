只需設定一次

安裝本機字幕核心

Chrome 擴充功能負責擷取分頁音訊與顯示字幕；語音辨識由你電腦上的 NVIDIA GPU 完成。

正在檢查本機核心…

請稍候。

1

下載 Windows 安裝器

安裝器很小，會從官方來源下載 FFmpeg、yt-dlp、CUDA Whisper 與字幕模型。

2

完成安裝與模型下載

Windows 可能顯示安全確認。依賴下載約 2.5–3 GB，安裝後約占 4 GB；時間取決於網路速度。

3

回來重新檢查

安裝完成後不必重新安裝擴充功能。

系統需求

  • Windows 10 或 Windows 11，Chrome／Edge／Vivaldi 116 以上
  • NVIDIA GTX／RTX 顯示卡與可用的 NVIDIA 驅動程式
  • 至少 5 GB 可用磁碟空間

哪些資料會離開電腦？

字幕辨識、音訊暫存與字幕快取都留在本機。安裝期間只會向 GitHub、Hugging Face 與 FFmpeg 發行來源下載工具及模型；Any Subtitle 不提供雲端 ASR 或遙測。
