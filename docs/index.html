Any Subtitle

Give videos without captions a local subtitle track.

Live captions

Capture audio from the tab you select and update captions continuously with the local GPU.

Accurate captions

Use accessible audio and a larger Whisper model, then synchronize the result to the player timeline.

Data stays local

No cloud speech recognition, ads, analytics, or telemetry. Accurate tracks are cached for seven days.

Windows Local Core required

The extension captures and displays captions. The free Any Subtitle Local Core runs FFmpeg, yt-dlp, and CUDA Whisper. Windows 10/11, an NVIDIA GTX/RTX GPU, and about 5 GB are required.
