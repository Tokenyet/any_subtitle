Any Subtitle Support

Start by checking the Local Core status.

Shows Not installed

Download and run Any Subtitle Local Core again, then return to the setup page and choose Check again.

Models are incomplete

Run the installer again to resume incomplete downloads. You do not need to reinstall the extension.

No captions

Confirm the video is playing and the tab has audio, then stop the current task and start again.

Supported environment

Windows 10/11, Chrome, Edge, or Vivaldi 116+, and NVIDIA GTX/RTX. DRM, browser internal pages, Picture-in-Picture, and canvas-only players are not guaranteed.
